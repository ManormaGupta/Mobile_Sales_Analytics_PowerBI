# Mobile Sales Analytics Dashboard 📊📱

A comprehensive Power BI Data Analytics project focused on exploring and transforming raw smartphone retail transaction data into actionable business insights.

## 📸 Dashboard Preview
![Mobile Sales Dashboard Preview](Screenshot%202026-05-21%20134044.png)

## 🔍 Project Overview & Objectives
In the fast-paced consumer electronics retail market, understanding sales performance, transaction formats, and regional demand shifts is critical for inventory optimization. This project delivers an interactive, end-to-end business dashboard designed to help retail managers track operational health and customer behavior.

## 📊 Core KPIs & Visuals Maintained
* **Financial Scale:** Explored metrics across Total Sales revenue (₹769.2M), Total Units Sold (19K), Average Sells Price, and total transaction volume.
* **Geographic Distribution:** Created map visuals to highlight revenue hubs across different tiers and regions.
* **Customer Feedback Loop:** Built a targeted Funnel chart to look closely at customer ratings.
* **Fintech Trends:** Used detailed Pie Charts to map out user preferences across Payment Methods (UPI vs. Credit/Debit Cards vs. Cash).

## 🛠️ Tech Stack & Methods Used
* **Data Presentation:** Power BI Desktop
* **Data Extraction & Transformation:** Power Query Editor
  * Handled multi-layered null value filtering.
  * Executed deep blank row removals to clean visual calculations.
* **Data Modeling:** Developed explicit date tables to manage weekly and monthly trend slicing seamlessly.

## 💡 Key Business Recommendations
1. **Optimize Fintech Promotions:** Because digital transaction modes like UPI capture a dominant chunk of transaction metrics, structuring strategic tie-ups with payment gateway providers can directly optimize checkout processing costs.
2. **Targeted Inventory:** Supply chain and distribution streams should prioritize high-value flagship smartphone units near metro regional centers where average transaction tickets consistently peak.
